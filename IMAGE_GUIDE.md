## Photography Tips

### For Best Results:

1. **Lighting**: Use natural daylight when possible. Avoid harsh shadows.

2. **Composition**:
   - Follow the rule of thirds
   - Include some negative space
   - Show texture and detail

3. **Food Photography**:
   - Shoot from a 45-degree angle for most dishes
   - Use overhead shots for flat lays and full table spreads
   - Include props like wine glasses, cutlery, or fresh herbs

4. **Interior Shots**:
   - Capture the warm atmosphere
   - Show the Thonet chairs and wooden furniture
   - Include the brick walls and warm lighting
   - Avoid empty tables if possible

5. **Catering Displays**:
   - Show variety and abundance
   - Use clean, white serving dishes for elegance
   - Arrange items symmetrically for professional look

## Image Optimization

Before uploading images:

1. **Resize** to recommended dimensions
2. **Compress** using tools like TinyPNG or ImageOptim
3. **Format**: Use JPG for photos (better compression)
4. **Quality**: Aim for 80-85% quality for good balance

## Fallback Display

If images are not yet added, the website will display:
- Placeholder colored backgrounds
- Icon representations
- Filename text

The site remains functional without images, but adding them creates the full experience.

## Testing Your Images

After adding images:

1. Clear your browser cache
2. Reload the website
3. Check each page:
   - Home page (hero, about section, highlights, wine section)
   - Catering page (header and 8 gallery images)
4. Test on mobile devices to ensure images look good at all sizes

## Need Help?

- Ensure images are named exactly as specified (case-sensitive)
- Check that images are in `public/images/` not `src/images/`
- Verify file extensions are `.jpg` not `.jpeg`
- Make sure images aren't too large (under 500KB each is ideal)
